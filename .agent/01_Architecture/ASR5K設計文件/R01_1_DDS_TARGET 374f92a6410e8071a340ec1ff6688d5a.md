# R01_1_DDS_TARGET

DDS 功能說明:
1. 要能設定**啟動**、**停止**的功能
2. 要能設定**頻率**的功能
3. 要能設定**振幅**的功能
4. 要能設定**偏移**的功能
5. 要能設定**振幅斜坡**的功能
5.1 要能設定**振幅斜坡上數時間**的功能
5.2 要能設定**振幅斜坡下數時間**的功能
5.3 要能設定**啟用振幅斜坡**的功能
6. 要能設定**偏移斜坡**的功能
6.1 要能設定**偏移斜坡上數時間**的功能
6.2 要能設定**偏移斜坡下數時間**的功能
6.3 要能設定**啟用偏移斜坡**的功能
7. 要能設定**頻率斜坡**的功能
7.1 要能設定**頻率斜坡上數時間**的功能
7.2 要能設定**頻率斜坡下數時間**的功能
7.3 要能設定**啟用頻率斜坡**的功能
8. 要能設定**啟動延遲**的功能
8.1 要能設定**啟動延遲時間**的功能
8.2 要能設定**啟動延遲啟用**的功能
9. 要能設定**停止延遲**的功能
9.1 要能設定**停止延遲時間**的功能
9.2 要能設定**停止延遲啟用**的功能
10. 要能設定**相位**的功能
10.1 要能設定**起始相位**的功能
10.2 要能設定**結束相位**的功能

功能流程說明:
1. 當 **啟動** 指令輸入時， DDS 會進入 **啟動延遲** 的狀態，如果 **啟動延遲** 結束， DDS 會直接套用 **起始相位**，進入 **運行** 的狀態，在 **運行** 的狀態下，開始執行 **振幅斜坡** **偏移斜坡** **頻率斜坡** 的控制:

**特殊模式 (DC Mode)**: 若設定 **頻率** 為 0，則進入 DC 模式。在此模式下，**振幅** 無作用 (視為 0)，僅 **偏移** 有作用。

**振幅斜坡**的控制，是要檢查 **啟用振幅斜坡**，如果是開啟的狀態，就要加振幅由零開始執行到目標值，如果是關閉的狀態，就要直接套用目標值。
**偏移斜坡**的控制，是要檢查 **啟用偏移斜坡**，如果是開啟的狀態，就要加偏移由零開始執行到目標值，如果是關閉的狀態，就要直接套用目標值。
**頻率斜坡**的控制，是要檢查 **啟用頻率斜坡**，如果是開啟的狀態，就要加頻率由零開始執行到目標值，如果是關閉的狀態，就要直接套用目標值。

1. 當 **停止** 指令輸入時， 就會檢查 **啟動振幅斜坡** 是否有開啟，如果是開啟的狀況，就要將振幅由目標值遞減至零，當振幅為零時，就要重置 **偏移斜坡** 為中心值32768，**頻率斜坡** 為初始值0，如果 **啟動振幅斜坡** 關閉的狀態， DDS 會進入 **停止延遲** 的狀態，如果 **停止延遲** 結束， DDS 會進入 **停止** 的狀態。

參數規定:
1. 時間範圍 0.000 秒 ~ 60.000 秒
2. 相位範圍 0.0 度 ~ 360.0 度
3. 頻率範圍:
- 0 Hz: DC 模式 (振幅無效，僅偏移有效)
- 1.00 Hz ~ 1000.00 Hz: AC 模式
4. 振幅範圍 0 ~ 65535，相對於 0~1PU
5. 偏移範圍 0 ~ 65535，相對於-1PU~1PU，中心點為32768
