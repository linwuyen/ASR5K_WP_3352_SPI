可以用兩種方式執行：

- CCS Expressions：操作 `spiA_master.stApp`
- 程式碼：呼叫固定 SPIA wrapper API

執行前確認主迴圈持續呼叫：

```c
runSPIBslave();
runSPIAmaster();
```

**1. 單筆讀取**

CCS Master 設定：

```text
spiA_master.stApp.u16TestAddr = 0x0400
spiA_master.stApp.u16CmdRead  = SPI_TEST_CMD_TRIGGER
```

或程式碼：

```c
readSPIAmaster(C2000_Version_spi_addr, callback);
```

Slave 檢查：

```text
g_u32DebugLastValidResponse = 0x051A8A90
```

Master 成功後：

```text
spiA_master.stApp.u16TestData = 0x8A90
spiA_master.stApp.u16CmdRead  = SPI_TEST_CMD_IDLE
```

---

**2. 單筆寫入**

建議使用真正可寫入的 `0x0900`：

```text
spiA_master.stApp.u16TestAddr = 0x0900
spiA_master.stApp.u16TestData = 0x0001
spiA_master.stApp.u16CmdWrite = SPI_TEST_CMD_TRIGGER
```

或程式碼：

```c
writeSPIAmaster(Output_ON_OFF_spi_addr, 1U, callback);
```

Slave 檢查：

```text
OUTPUT_ON       = 1
g_u32DebugLastValidResponse = 0x09010001
```

Master 成功後：

```text
spiA_master.stApp.u16CmdWrite = SPI_TEST_CMD_IDLE
```

---

**3. 一般 Block 傳輸**

CCS 內建 16 點順序寫入測試：

```text
spiA_master.stApp.u16CmdSeqWriteTest = SPI_TEST_CMD_TRIGGER
```

Master 會傳送：

```text
0x3000 ～ 0x300F：16 點資料
0x3FFF：Block End
```

程式碼自訂 Block：

```c
writeBlockSPIAmaster(txBuffer, rxBuffer, length, callback);
```

Slave 檢查：

```text
g_u16SpiBlockRam[0] ～ [15]
spiB_slave.u16BlockWriteIndex = 16
spiB_slave.u16BlockExpectedLen = 16
spiB_slave.u16BlockStatus = 3        // READY
spiB_slave.u16BlockReady = 1
spiB_slave.u32BlockRxCount
```

---

**4. 4095 點波形 Block 傳輸**

Ramp 波形：

```text
spiA_master.stApp.u16CmdWave4095Test = SPI_TEST_CMD_TRIGGER
```

Sine 波形：

```text
spiA_master.stApp.u16CmdSineWave4095Test = SPI_TEST_CMD_TRIGGER
```

或程式碼：

```c
writeWaveBlockSPIAmaster(0U, callback); // Ramp
writeWaveBlockSPIAmaster(1U, callback); // Sine
```

Slave 檢查：

```text
g_u16SpiBlockRam[0] ～ [4094]
spiB_slave.u16BlockWriteIndex = 4095
spiB_slave.u16BlockExpectedLen = 4095
spiB_slave.u16BlockStatus = 3
spiB_slave.u16BlockReady = 1
spiB_slave.u16BlockProgress = 100
```

Ramp 預期值：

```text
g_u16SpiBlockRam[0]    = 0x0010
g_u16SpiBlockRam[1]    = 0x0020
g_u16SpiBlockRam[4094] = 0xFFF0
```

Master 共通檢查：

```text
spiA_master.stApp.u32StressPassCnt
spiA_master.stApp.u32StressFailCnt
spiA_master.stDiag.u16LastErrType
spiA_master.stDiag.u16LastErrRxAddr
spiA_master.stDiag.u16LastErrRxData
```
